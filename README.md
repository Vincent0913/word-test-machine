## 創作動機：
　　
英文對於現代學生來說非常重要，因為這樣，我們有很多單字要背。
在背單字的時候，想到可以用反覆測驗的方式來增加記憶，於是我開發了單字測驗機，讓使用者能夠自訂題庫測驗，像是4000單、商業單字、醫學單字。
